# Docker Pht

Minimal [Docker][1] container based on [Alpine Linux][2] with [PHP 7.2][3] and [Pht threading extension][4].

## HowTo

Edit
`./srv/app.php`.

Run
`docker-compose up` .

[1]: https://www.docker.com/
[2]: https://alpinelinux.org/
[3]: https://secure.php.net/
[4]: https://github.com/tpunt/pht
